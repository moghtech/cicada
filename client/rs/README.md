# Cicada

Docs: [https://docs.rs/cicada_client/latest/cicada_client](https://docs.rs/cicada_client/latest/cicada_client).

This is a client library for the Cicada Core API.
It contains:
- Definitions for the application [api](https://docs.rs/cicada_client/latest/cicada_client/api/index.html)
	and [entities](https://docs.rs/cicada_client/latest/cicada_client/entities/index.html).
- A [client](https://docs.rs/cicada_client/latest/cicada_client/struct.CicadaClient.html)
	to interact with the Cicada Core API.
- Information on configuring Cicada
	[Core](https://docs.rs/cicada_client/latest/cicada_client/entities/config/core/index.html) and
	[Periphery](https://docs.rs/cicada_client/latest/cicada_client/entities/config/periphery/index.html).
